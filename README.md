# SEGUROS-CARACAS

Aplicación web para generar informes periciales de aseguradoras en formato PDF.

## Instalación y ejecución

1. Navega al directorio del proyecto:
   ```bash
   cd informe-pericial-aseguradora
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el entorno de desarrollo:
   ```bash
   npm run start
   ```

## Estructura

- `informe-pericial-aseguradora/` – aplicación principal con el formulario y scripts.
- `README.md` – este documento.
