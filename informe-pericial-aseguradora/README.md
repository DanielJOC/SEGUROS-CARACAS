# Informe Pericial Aseguradora

Este proyecto es una aplicación web para la gestión de informes periciales de aseguradoras. Permite a los usuarios ingresar datos relevantes y generar un informe en formato PDF.

## Estructura del Proyecto

El proyecto tiene la siguiente estructura de archivos:

```
informe-pericial-aseguradora
├── src
│   ├── index.html          # Página principal con el formulario
│   ├── css
│   │   └── styles.css      # Estilos CSS para la interfaz
│   ├── js
│   │   ├── main.js         # Lógica principal de JavaScript
│   │   └── pdf-generator.js # Funciones para generar PDF
│   └── assets              # Recursos adicionales (imágenes, etc.)
├── package.json            # Configuración de npm
└── README.md               # Documentación del proyecto
```

## Instalación

1. Clona el repositorio en tu máquina local.
2. Navega al directorio del proyecto.
3. Ejecuta `npm install` para instalar las dependencias necesarias.

## Uso

1. Abre `src/index.html` en tu navegador.
2. Completa el formulario con los datos requeridos.
3. Haz clic en el botón para generar el informe en PDF.

## Dependencias

Este proyecto utiliza las siguientes dependencias:

- jsPDF: Para la generación de archivos PDF.
